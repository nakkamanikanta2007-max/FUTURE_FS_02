# 🚀 LeadSphere CRM

A modern Customer Relationship Management (CRM) system developed as part of the Future Interns internship program.

---

## 📌 Project Overview

LeadSphere CRM helps businesses manage customer leads efficiently. Users can add, edit, delete, search, filter, and track leads through a modern dashboard.

---

## ✨ Features

- 🔐 User Login
- 📊 Dashboard
- 👥 Lead Management (CRUD)
- 🔍 Search Leads
- 🎯 Filter Leads by Status
- 📈 Analytics Charts
- 📅 Follow-Up Tracking
- 📄 Export Leads to CSV
- 🔔 Toast Notifications
- 📱 Responsive User Interface

---

## 🛠️ Technologies Used

- React.js
- Vite
- Express.js
- Tailwind CSS
- Axios
- React Router
- React Toastify
- Chart.js
- JSON File Storage

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/nakkamanikanta2007-max/FUTURE_FS_02.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
